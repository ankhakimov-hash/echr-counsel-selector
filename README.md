# ECHR Counsel Selector

Одностраничный desktop-прототип для кейса 4 из тестового задания: выбор консультантов для жалобы в ЕСПЧ после проигрыша в Верховном суде Украины, когда часть материалов засекречена.

## Что умеет

- сравнивает консультантов колонками;
- учитывает опыт ЕСПЧ, санкционный кейс-фит, Chambers / Legal 500, работу с закрытыми материалами, цену и оплату;
- пересчитывает рейтинг через слайдеры весов;
- фильтрует по бюджету, банковской оплате и конфликтным рискам;
- добавляет нового консультанта как новую колонку сравнения;
- принимает ссылку на сайт фирмы или профиль консультанта и создает AI-import черновик колонки;
- позволяет редактировать любую ячейку матрицы прямо в таблице;
- выгружает текущую таблицу в Excel (`.xls`);
- открывает печать браузера для сохранения в PDF.

AI-import в этой статической версии работает как UX-прототип: известные домены распознаются по встроенным профилям, неизвестные сайты превращаются в черновик с ссылками на Chambers, Legal 500 и поиск по СМИ. Для настоящего автоматического парсинга нужен backend или API.

## Стартовые ориентиры

- Россия: ALRUD;
- Украина: Asters;
- Европа: Doughty Street Chambers.

Оценки внутри прототипа демонстрационные. Перед реальным выбором нужны fresh Chambers / Legal 500 check, conflict check, санкционный скрининг, письменный fee quote и подтверждение доступности команды.

## Как запустить

Откройте `index.html` в браузере.

## Как выложить на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите в него `index.html`, `style.css`, `app.js` и `README.md`.
3. В настройках репозитория откройте Pages.
4. Выберите публикацию из ветки `main`, папка `/root`.
